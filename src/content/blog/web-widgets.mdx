# 访问量统计

![访问量统计](https://origin.picgo.net/2025/09/13/image6a1c3edd2f6c6cbe.png)

我们这里用的是不蒜子访问量统计

要使用不蒜子必须在页面中引入busuanzi.js，目前最新版如下：

```html
<script async src="//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js">
</script>
```

要显示站点总访问量，复制以下代码添加到你需要显示的位置。有两种算法可选：

算法a：pv的方式，单个用户连续点击n篇文章，记录n次访问量

```html
<span id="busuanzi_container_site_pv">
    本站总访问量<span id="busuanzi_value_site_pv"></span>次
</span>
```

算法b：uv的方式，单个用户连续点击n篇文章，只记录1次访客数

```html
<span id="busuanzi_container_site_uv">
  本站访客数<span id="busuanzi_value_site_uv"></span>人次
</span>
```

# 随机诗词

![随机诗词](https://origin.picgo.net/2025/09/13/4f3dfa99c7ff1ff82.png)

<LinkCard
  title="今日诗词"
  desc="awa"
  url="https://www.jinrishici.com/"
  img="https://www.jinrishici.com/img/logo.png"
/>

我们可以调用 `今日诗词API` ，这是一个可以返回一句古诗词名句的接口。

直接在要添加的页面加上下面的代码：

```html
<script src="https://sdk.jinrishici.com/v2/browser/jinrishici.js" charset="utf-8"></script>
    <p class="text-xl text-base-content/80">
        <span id="jinrishici-sentence" class="font-bold">正在加载今日诗词....</span>
    </p>
```

我这个文字加粗了，可以把 ` class="font-bold"` 去掉以去掉加粗。
